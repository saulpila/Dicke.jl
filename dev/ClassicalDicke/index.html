Dicke.ClassicalDicke.ClassicalDickeSystem — Type
struct ClassicalDickeSystem <: ClassicalSystems.ClassicalSystem

Instance of ClassicalDicke.ClassicalDickeSystem which represents the classical Dicke model with the given parameters $ω_0$, $ω$, and $γ$. See Eq. (5) of Ref. [10]. To generate this struct, use the constructor

ClassicalDickeSystem(;ω₀::Real,ω::Real,γ::Real)

For example, system = ClassicalDickeSystem(;ω₀=1, ω=1, γ=1).

This struct may be passed to all functions in this module that require an instance of ClassicalDicke.ClassicalDickeSystem, as well as functions in other modules that require the abstract ClassicalSystems.ClassicalSystem, such as ClassicalSystems.integrate.

Dicke.ClassicalDicke.classicalPathRandomSampler — Method
function classicalPathRandomSampler(system::ClassicalDickeSystem;ϵ,dt=3)

This function returns a function sample(), which produces random points within the classical energy shell at energy ϵ. This only works at energy shells where the classical dynamics are ergodic. The function sample() returns points from a fixed chaotic trajectory picked at random separated by a fixed time interval dt.

Arguments:

  • system should be generated with ClassicalDicke.ClassicalDickeSystem.
  • ϵ is the scaled energy $ϵ=E/j$ of the energy shell from where to sample.
  • dt is the fixed time interval that separates the points that are returned by sample().

Dicke.ClassicalDicke.density_of_states — Method
function density_of_states(system::ClassicalDickeSystem;j,ϵ)

Returns the semiclassical density of states (DoS) $ν(ϵ)$, in units of $1/ϵ$. This is computed with an expression similar to Eq. (19) of Ref. [2], where we add an additional factor of $j$ to have units of $1/ϵ$ instead of $1/E$, and the integral is performed with a change of variable.

Arguments:

  • system should be generated with ClassicalDicke.ClassicalDickeSystem.
  • j is the value of $j$
  • ϵ is the scaled energy $ϵ=E/j$

See Plotting the density of states for an example.

Dicke.ClassicalDicke.q_of_ϵ — Method
function q_of_ϵ(system::ClassicalDickeSystem;Q,P,p,ϵ,signo::Union{typeof(-),typeof(+)}=+,returnNaNonError=true)

Returns the solutions $q_\pm$ of the second degree equation in $q$ given by

\[ h_\text{cl}(Q,q,P,p)=\epsilon,\]

where $h_\text{cl}$ is given by Eq. (5) of Ref. [10].

Arguments:

  • system should be generated with ClassicalDicke.ClassicalDickeSystem.
  • Q, P, p, and ϵ are values of $Q$, $P$, $p$, and $\epsilon$, respectively.
  • signo is + for $q_+$ and - for $q_-$
  • If returnNaNonError is true, then NaN is returned if there are no solutions. If it is false, and error is raised.

Dicke.ClassicalDicke.q_sign — Function
function q_sign(system::ClassicalDickeSystem,x,ϵ=hamiltonian(system)(x))

Returns the sign of the root of the second degree equation in $q$ given by

\[ h_\text{cl}(Q,q,P,p)=\epsilon.\]

That is, this function returns + if q=x[2] ≈ q_of_ϵ(system;Q=x[1],P=x[3],p=x[4],ϵ=ϵ,signo=+) and returns - if q=x[2] ≈ q_of_ϵ(system;Q=x[1],P=x[3],p=x[4],ϵ=ϵ,signo=-).

Arguments:

  • system should be generated with ClassicalDicke.ClassicalDickeSystem.
  • x is a vector in the form [Q,q,P,p].
  • ϵ should be the energy of x. If this is not passed, it is computed using hamiltonian(system).
